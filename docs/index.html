Introduction

A simple npm package for translating between English and Swahili. This tool offers quick and accurate translations, complete with word meanings, grammatical types, and usage examples. Whether you're a developer building multilingual applications, a language learner, or someone working with Swahili and English content, Swahili-English Translator provides an efficient solution for your translation needs. Let's explore how to integrate and utilize this versatile package in your projects."

API Reference ğŸ“š

translateEng(word)

Translates an English word to Swahili.

  • Parameters:
    • word (string): The English word to translate
  • Returns: An object containing the Swahili translation, word type, and an example

translateKsw(word)

Translates a Swahili word to English.

  • Parameters:
    • word (string): The Swahili word to translate
  • Returns: An object containing the English translation, word type, and an example
